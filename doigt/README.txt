###   Application qui compte le nombre de doigts leves d'une main   ###
apres avoir installer python et toutes les dependances,
ouvrir le fichier Main.py et configurer la source de la camera ( 0 pour la webcam , l'adresse ip pour une camera sans fil)
perso j'utilise l'application android ip cam pour la camera ip. je conseille une configuration de resolution video d'au plus 1280x720 pour ne pas encombrer l'ecran
lancer l'application grace a la commande python Main.py
